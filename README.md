# Premium Portfolio v3

A premium animated freelance portfolio for Adeel Iqbal Anjum, built with Next.js, React, TypeScript, Tailwind CSS, dark/light mode, smooth scrolling, glassmorphism cards, and polished portfolio sections.

## Tech stack

- Next.js App Router
- React + TypeScript
- Tailwind CSS
- GSAP-ready animation structure
- Responsive dark/light UI

## Getting started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Build

```bash
npm run build
npm run start
```

## Notes

Generated files such as `.next`, `node_modules`, `*.tsbuildinfo`, and build artifacts are excluded from the repository.
